## 🔭 Project Overview
**Across Time: AI Chronicles 2581** is a terminal-based interactive fiction game written in Python. As a time traveler arriving in a highly advanced future, you must team up with a friendly virtual assistant (ARIVA) to stop a corrupted global intelligence (NEXUS) from enslaving humanity. 

## ✨ Key Features
* **Branching Narrative:** Your choices dictate the fate of the future.
* **Immersive CLI:** Features retro typewriter text effects and ASCII art.
* **Logic & Cryptography Puzzles:** Crack ciphers, decode Morse, and identify patterns.
* **Ethical Debugging:** Locate and fix the "bugs" in the rogue AI's moral code.

## 🧠 The AI Ecosystem
The year 2851 is governed by specialized networks:
* 🏥 **MedAI (HEALER-12):** Planetary health and lifespan optimization.
* 🚗 **VechAI (TRANSIT-9):** Global zero-emission autonomous transit.
* 🎓 **TeachAI (Professor LEARN-7):** Personalized neural-pattern education.
* 👁️ **NEXUS:** The central fusion system that malfunctions, triggering the crisis.

## 🕹️ Gameplay & Challenges
Players must survive the "Great Malfunction" through rapid problem-solving:
1. **Temporal Verification:** Reverse-string identity checks.
2. **Cipher Cracking:** Alphanumeric sequence puzzles to bypass firewalls.
3. **Signal Interception:** Translating Morse code distress signals.
4. **Code Auditing:** Finding the exact line of Python code causing NEXUS to go rogue.

## 📚 Themes & Learning Outcomes
* **Themes:** The AI alignment problem, technological utopias, and the necessity of human oversight.
* **Concepts:** Python control flow, error handling, string manipulation, and terminal output pacing.

## 💻 Tech Stack
* **Language:** Python 3.x
* **Core Libraries:** `sys`, `time`
* **Architecture:** Procedural logic flow with robust `try/except` error handling.
